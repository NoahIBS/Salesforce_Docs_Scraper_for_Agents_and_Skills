# Other Data 360 Features for Marketers

Marketing Cloud Next includes other Data 360 features that can help you with segments and reporting.

### Required Editions

Available in: Salesforce **Enterprise** and **Unlimited** Editions with Marketing Cloud Next **Growth** or **Advanced** Edition  
---  
Feature | Description  
---|---  
[Bring Your Own Model](https://help.salesforce.com/s/articleView?id=data.c360_a_bring_your_own_model.htm&language=en_US&type=5) | Build, train, and explore a data model with the Data 360 Python connector, which moves data between your prediction and a Data Model Object (DMO). This connector is built on [Query API](https://developer.salesforce.com/docs/atlas.en-us.api_rest.meta/api_rest/resources_query.htm).  
[Calculated Insights](https://help.salesforce.com/s/articleView?id=data.c360_a_calculated_insights.htm&language=en_US&type=5) | Define and calculate multidimensional metrics on profile, segment, and population objects.  
[Data Actions](https://help.salesforce.com/s/articleView?id=data.c360_a_data_actions_cdp.htm&language=en_US&type=5) | Trigger actions with Salesforce Platform Events or webhooks using streaming insights and engagement data.  
[Data Explorer](https://help.salesforce.com/s/articleView?id=data.c360_a_data_explorer.htm&language=en_US&type=5) | Validate data lake objects, data model objects, and calculated insights to ensure that data and formulas are accurate.  
[Data Graphs](https://help.salesforce.com/s/articleView?id=data.c360_a_data_graphs.htm&language=en_US&type=5) | Combine and transform normalized table data from data model objects (DMOs) into new, materialized views of your data.  
[Data Shares](https://help.salesforce.com/s/articleView?id=data.c360_a_data_shares.htm&language=en_US&type=5) | Give third-party partners access to your data, so that they can build powerful analytical insights and machine learning models.
