# Set Up Lead Assignment Rules

After a prospect is converted to a lead, lead assignment rules determine how the record owner is assigned. To assign a different record owner for leads, you can create a new rule.

[](https://help.salesforce.com/s?language=en_US)

  1. From Setup, enter `Assignment Rules` in the Quick Find box, then select **Lead Assignment Rules**.
  2. Select **New** and give the rule a name. Specify that you want this rule to be an active rule for leads, and then click **Save**.
  3. To create the rule entries, click **New**.
  4. Specify the order in which the rule entry is processed, the condition the lead must match, and the user the lead is assigned to.
