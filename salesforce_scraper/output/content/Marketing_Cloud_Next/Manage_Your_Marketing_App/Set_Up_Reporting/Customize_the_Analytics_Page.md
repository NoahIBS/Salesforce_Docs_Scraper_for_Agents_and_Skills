# Customize the Analytics Page

To help users find and interact with Marketing Cloud Next reports and dashboards, create collections and filters.

### Required Editions

Available in: Salesforce **Enterprise** and **Unlimited** Editions with Marketing Cloud Next **Growth** or **Advanced** Edition  
---  
User Permissions Needed  
---  
To customize the Analytics tab: | Marketing Cloud Admin permission setORMarketing Cloud Manager permission set  
  
  1. To simplify how users find reports and dashboards, create a collection on the Analytics tab.
     1. In Marketing Cloud Next, open the Analytics tab.
     2. In the Collections widget, click **+** , and give the collection a name.
     3. Search and select the reports and dashboards to include in the collection.
  2. To further customize the dashboards, add filters.
     1. From the Analytics tab, open a dashboard, and then click **\+ Filter** in the toolbar.
     2. Select a field to filter, and give the filter a name.
     3. Add filter values as needed. When you finish, click **Apply**.
     4. To save the filter, click **Add** , and then save the dashboard.
