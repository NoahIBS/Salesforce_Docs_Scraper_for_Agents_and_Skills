# Set Up Agentforce for Business Units

Agentforce in Marketing Cloud Next helps you create campaign briefs, campaigns, message content, and more using Einstein technology. Einstein predictive features provide insights and data to inform the content and timing of your marketing activities.

### Required Editions

Available in: Salesforce **Enterprise** and **Unlimited** Editions with Marketing Cloud Next **Advanced** Edition  
---  
User Permissions Needed  
---  
To set up Agentforce for business units: | Marketing Cloud Admin permission set  
  
See [Enable Einstein Features in Marketing Cloud Next](https://help.salesforce.com/s/articleView?id=mktg.mktg_admin_setup_einstein.htm&language=en_US&type=5) and [Explore Agentforce and Einstein in Marketing Cloud Next](https://help.salesforce.com/s/articleView?id=mktg.mktg_einstein_learning_map.htm&language=en_US&type=5).

Note Einstein Metrics Guard isn’t supported with business units.
