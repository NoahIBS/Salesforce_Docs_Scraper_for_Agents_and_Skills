# Manage Campaigns and Flows with Marketing Calendar

Track and manage your upcoming or ongoing campaigns and segment flows effortlessly from the Marketing Calendar.

### Required Editions

Available in: Salesforce **Enterprise** and **Unlimited** Editions with Marketing Cloud Next **Growth** or **Advanced** Edition  
---  
User Permissions Needed  
---  
To access Marketing Calendar | Marketing Cloud Admin permission setORMarketing Cloud Manager permission setORAccess Marketing Calendar user permission, for Sales users  
  
Marketing Calendar provides a centralized hub for marketing teams to plan and organize their campaigns and flows. By default, you can see the Campaigns, Campaign Segment Flows, and Segment Flows calendars. You can only hide these calendars but can’t delete or customize the colors.

You can perform these tasks on the calendar:

[](https://help.salesforce.com/s?language=en_US)

  * Organize your campaigns and flows in daily, weekly, and monthly views. 
  * Click a calendar event to view detailed information such as type, status, start date, segments, population, and so on. 
  * Manage campaigns and flows directly from the calendar. 
  * Create a custom object calendar.
  * Add new events to the calendar and link them to campaigns.
  * Drag a campaign to change its date. However, you can’t drag the segment flows to change the date.



[](https://help.salesforce.com/s?language=en_US)

## Access Marketing Calendar

You can access the Marketing Calendar from the Marketing Cloud Next home page, the Marketing Calendar tab, or the App Launcher. The Marketing Calendar is integrated with the Sales Calendar.

Campaigns and flows can have record-level sharing enabled, and you can only see the campaigns and flows to which you have access. If you lack access to a specific field displayed on the calendar hover popover, you’ll see the field names but can’t view any data for those fields. For example, you'll see an empty field if you don't have access to the Start Date field for campaigns.

[](https://help.salesforce.com/s?language=en_US)

## Create a Custom Object Calendar

The Marketing Calendar allows you to create a custom object calendar that you can edit, delete, and personalize with different colors. To create a custom object calendar, see [Create an Object Calendar in Lightning Experience](https://help.salesforce.com/s/articleView?id=sales.calendar_create.htm&language=en_US&type=5) and [Considerations for Calendar created from Salesforce Custom Objects](https://help.salesforce.com/s/articleView?id=sales.calendar_create_limitations.htm&language=en_US&type=5).

[](https://help.salesforce.com/s?language=en_US)

## Manage List Views

The list view filters include Campaign Flows and Standalone Flows, enabled for the Marketing Calendar and correspond to the Campaign Flows and Segment Flows calendars. To view these list views, go to the Flows tab in the Marketing app or Flows Object Home in the Automation app.

The list views filters display flows triggered by segments, forms, and events, but events and form-triggered flows don’t appear on the Marketing Calendar because they don’t have a specific start date. These list views filters are created by default, and you can’t edit or delete them.

  * **[Add an Event to the Marketing Calendar](https://help.salesforce.com/s/articleView?id=mktg.mktg_add_event_to_calendar.htm&language=en_US&type=5)**  
Track important events by adding them to your calendar. You can also associate these events with campaigns.
