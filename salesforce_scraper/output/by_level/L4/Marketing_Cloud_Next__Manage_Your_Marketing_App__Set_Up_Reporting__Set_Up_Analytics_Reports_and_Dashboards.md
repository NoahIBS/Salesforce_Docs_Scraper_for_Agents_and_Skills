# Set Up Analytics Reports and Dashboards

Marketing analytics packages include preconfigured dashboards and folders with reporting data about your marketing efforts.

### Required Editions

Available in: Salesforce **Enterprise** and **Unlimited** Editions with Marketing Cloud Next **Growth** or **Advanced** Edition  
---  
  
This table lists the main analytics packages that you install to access reporting tools. The Setup label is used on marketing setup pages, and the package names appear on the installation and data stream deployment pages.

To install the Marketing Analytics package, you need at least two available dynamic dashboard licenses. If you've reached the limit, reduce the number of dynamic dashboards in use or purchase more licenses, then retry the installation. See [Find the Dynamic Dashboards in my Organization](https://help.salesforce.com/s/articleView?id=000385369&language=en_US&type=1).

Setup Label | Package Name | Details  
---|---|---  
Marketing Engagement Analytics  | Marketing Analytics | This package contains these Analytics folders for dashboards and reports: [](https://help.salesforce.com/s?language=en_US)

  * Email Engagement Dashboard
  * Email Engagement Dashboard V2
  * Email Engagement Reports
  * Email Engagement Reports V2

Both dashboards are dynamic, and you can add users to each folder on the Analytics tab.   
Flow Reports Analytics | Salesforce Data 360 \- Flow Reports | This package contains these Analytics folders for dashboards and reports: [](https://help.salesforce.com/s?language=en_US)

  * Flow Reports

You can add users to each folder on the Analytics tab.   
SMS Analytics  | Marketing Analytics - SMS | This package contains these Analytics folders for dashboards and reports: [](https://help.salesforce.com/s?language=en_US)

  * SMS UMA Dashboards 
  * SMS UMA Reports 

This package is available only when using the SMS add-on.  
Landing Pages and Forms Analytics | Marketing Analytics - Landing Pages and Forms | This package contains these Analytics folders for dashboards and reports: [](https://help.salesforce.com/s?language=en_US)

  * Forms Engagement Dashboards 
  * Forms Engagement Reports 
  * Landing Page Engagement Dashboards 
  * Landing Page Engagement Reports 

Before you install this package, add Data 360 integrations to the default Marketing Landing Pages site.  
  
  * **[Install and Connect Analytics Objects](https://help.salesforce.com/s/articleView?id=mktg.mktg_admin_install_analytics.htm&language=en_US&type=5)**  
Install the analytics packages and add users to analytics folders. Then, head to Digital Experiences to finalize the data connection.
